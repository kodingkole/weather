Weather App



This is a simple \*\*Weather App\*\* web application that displays current weather information for any city.



&#x20;Features

\- Search weather by city name

\- Display temperature, humidity, and weather description

\- Responsive and simple UI

\- Interactive icons for weather conditions



&#x20;Technology Stack

\- HTML

\- CSS

\- JavaScript

\- OpenWeatherMap API (for fetching weather data)
